# Consumers-Behavior-Analysis
Retail Consumer Behavior Analysis project that explores customer shopping patterns across demographics, discounts, reviews, seasons, and sales channels. Built using Python for data cleaning, SQL for analysis, and Power BI for visualization to deliver actionable insights for improving marketing, engagement, and sales performance.

# Overview

This project is an end-to-end data analytics solution designed to analyze consumer shopping behavior and extract actionable business insights. It demonstrates the complete data workflow — from data loading and cleaning in Python to SQL-based analysis and interactive dashboard creation in Power BI. The goal is to identify trends, customer segments, and key factors influencing purchasing decisions.

# Dataset Used

The dataset contains retail transaction data including customer demographics, product categories, purchase amounts, discounts, reviews, seasons, and sales channels (online vs. offline).
It is used to analyze buying patterns, customer loyalty, and revenue drivers.

# Tools & Technologies

Python (Pandas, NumPy, Matplotlib/Seaborn) – Data loading, cleaning, and EDA

SQL (PostgreSQL / MySQL / SQL Server) – Data modeling and analytical queries

Power BI – Interactive dashboard development and KPI visualization

Gamma – Presentation (PPT) creation

Git & GitHub – Version control and project documentation

The Power BI dashboard includes:

The analysis identifies key drivers of customer purchasing behavior, highlights high-performing segments, and provides recommendations to improve marketing effectiveness, customer engagement, and revenue
